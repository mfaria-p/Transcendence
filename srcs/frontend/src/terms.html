Projeto 42 Porto — Transcendence

Termos e Condições

Ao usar a plataforma Transcendence, concorda com os termos abaixo. Este projeto é académico e destina-se à avaliação do curriculum da 42 Porto.

1. Acesso e conta

  • O acesso requer uma conta fornecida para fins académicos. Guarde as suas credenciais em segurança.
  • Não partilhe as credenciais nem tente aceder à conta de terceiros.

2. Uso aceitável

  • Utilize a plataforma apenas para fins académicos e demonstrativos.
  • É proibido explorar vulnerabilidades, realizar engenharia reversa ou perturbar o serviço.
  • Conteúdos ofensivos, ilegais ou que violem direitos de terceiros não são permitidos.

3. Dados e privacidade

  • Os dados recolhidos servem apenas para funcionamento do projeto (ex.: autenticação, perfis, partidas).
  • Não há fins comerciais. Os dados podem ser removidos ao término das avaliações.
  • Não partilhe informação sensível; use apenas o necessário para testar a aplicação.

4. Segurança

  • Comunique falhas de segurança à equipa do projeto antes de qualquer divulgação.
  • Não execute ataques de carga, spam ou qualquer forma de negação de serviço.

5. Conteúdos e propriedade

  • O código e os recursos são para uso académico. Não utilize materiais fora deste contexto sem autorização.
  • As marcas e logótipos usados pertencem aos seus respetivos titulares.

6. Limitação de responsabilidade

  • O serviço é fornecido "tal como está" para fins de avaliação académica.
  • Não há garantias de disponibilidade, integridade ou suporte contínuo.

7. Suspensão e remoção

  • Contas ou sessões podem ser suspensas em caso de abuso, violação destes termos ou exigência académica.
  • Os dados podem ser limpos a qualquer momento como parte do ciclo de desenvolvimento ou avaliação.

8. Contacto

Para dúvidas ou reporte de problemas, contacte a equipa do projeto Transcendence na 42 Porto.
